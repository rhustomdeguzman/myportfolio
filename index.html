Smart Helmet with Liqour Detector

Smart Helmet with Liqour Detector is a helmet capable of detecting if the driver is drunk. Preventing the user to use his/her motorcycle when the said helmet detects alcohol in his/her breathe is it's main objective. With that being said, once the device has confirmed that the user is drunk, it will send a notification to the family of the user to inform them his/her current state. The notification that will be sent is an email which contains the driver's alcohol content reading and his/her exact location.
